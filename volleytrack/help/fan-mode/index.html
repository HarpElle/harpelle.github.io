Spectator View

Fan Mode

Fan mode lets spectators follow a live match from their own phones using a match code or shared link.

Quick answer: Start broadcast from the live match, share the 6-character code, let fans join, and keep scoring normally. Fans see live score and activity while you stay in control of the official match state.

Fan mode

Share a code or link so fans can follow from the stands.

Start a broadcast

  1. Open the match you are scoring.
  2. Open the share or spectator option.
  3. Start the broadcast and confirm the match code.
  4. Share the code or QR/link with parents and fans.

What fans can see

  • Current set and score.
  • Recent match activity.
  • Structured cheers and alerts where enabled.
  • Completed-match context and optional recap features.

Coach control

The coach scorer remains the source of truth. Spectator score-correction alerts are suggestions; they do not change the official score until the scorer decides to correct it.

Free spectator note: Free active spectator viewing may include non-interruptive banner ads. Ads should not block live score, alerts, or recap generation.
