Troubleshooting and FAQ

Use these answers when something does not match the real match or a feature needs extra context.

Need help quickly? If the match is live, preserve the score first. Correct the score manually, use notes if needed, and fix detailed stats after the rally or set.

Voice input heard me but found no actions. What should I do?

Try a shorter phrase with a player number and one rally outcome. Example: "Number 7 kill from number 1." If the gym is too loud, use manual entry for that rally.

Can I substitute an ineligible player?

Yes, if the real lineup requires it. Ineligible players stay grouped separately to prevent mistakes, but you can confirm an override when correcting the match state.

I put a libero in the starting lineup as a normal player. Can I fix it?

Use the libero correction action from the substitute sheet when the player is in a valid back-row context. This is treated as a lineup correction, not a counted substitution.

Why did Fan mode not update instantly?

Live updates depend on network connectivity for the scoring device and the spectator device. Keep scoring locally; VolleyTrack should push updates when the connection recovers.

Do spectators change the official score?

No. Spectator alerts can notify the scorer, but the coach/scorer remains the source of truth.

How do I avoid bad data?

  • Confirm lineups before each set.
  • Use undo immediately for the last action.
  • Use score correction for older score mistakes.
  • Review voice actions before confirming.
  • Finish the match only after the final score is correct.

Still stuck?

Email jason@harpelle.com with the device, app version, match situation, and what you expected to happen.
