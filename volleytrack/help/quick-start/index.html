Quick Start

Use this guide when you are courtside and need the shortest reliable path to a usable match record.

Quick answer: Create or choose your team, start a match, confirm the lineup and first server, score each rally, use undo for mistakes, finish the match, then review the summary.

1. Choose your setup path

For a one-off match, use Quick Match. For a season, tournament, or repeated roster, create a season and event first so matches stay organized.

  1. Open VolleyTrack and choose Quick Match or an existing event.
  2. Enter your team name, opponent, and match format.
  3. Add or review the roster before starting the lineup.

2. Confirm the lineup

VolleyTrack uses rotation positions P1 through P6. P1 is the server when your team serves first. If your team uses liberos, mark them before play when possible.

  1. Place the six starting players in rotation order.
  2. Mark the active libero if one begins in the back row.
  3. Confirm the first server when prompted.

3. Score the match

The live screen is built around fast corrections. Score the point first, log stats as needed, and use undo immediately if the last action was wrong.

  • Tap your side or the opponent side for points.
  • Long-press or open stat controls for substitutions, timeouts, and corrections.
  • Use voice input for rally actions when you have time to review the confirmation screen.

4. Save and review

When the match ends, finish the match from the set modal or match menu. Review the summary for set scores, player stats, and optional AI analysis.

Do not rush finalization. Confirm the final score before finishing the match. If the score is wrong, correct the score before advancing.
