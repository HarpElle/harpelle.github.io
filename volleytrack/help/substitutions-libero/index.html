Lineup Corrections

Substitutions and Libero

Substitution logic keeps normal volleyball pairings fast while still allowing the coach to correct real-world lineup surprises.

Quick answer: Tap an occupied live court card to select that player for stats. Long-press an occupied card to open the substitute sheet. Regular substitutions count and create pairings after the score is no longer 0-0; libero replacements and corrections do not count against substitutions.

Normal substitutions

  1. Long-press the on-court player or live court slot to open the substitute sheet. Tapping an empty slot also opens the sheet.
  2. Choose the eligible player entering the match.
  3. Confirm the substitution and continue scoring.

After a regular non-libero substitution, VolleyTrack pairs the players so the likely return sub is easy to find later. Regular substitutions after the set is no longer 0-0 reduce the substitution counter for the set.

Ineligible players

VolleyTrack keeps likely-ineligible players grouped separately so accidental taps are less likely. A player may appear there because they are already on court, a libero would be in the front row, another libero is already on court, or the normal sub-pairing logic does not match the real situation.

If the real match requires it, expand the ineligible group, choose the player, and confirm the override.

Use overrides deliberately. Substituting an ineligible player is available for correcting reality, rule differences, injury substitutions, or changed lineups, not for routine lineup flow.

Libero workflow

Liberos use an alternate live card color so they are easy to spot. Known liberos are blocked from front-row entries, and VolleyTrack keeps the active court to one libero at a time.

  • Libero replacements do not decrement the substitution count.
  • When marking a player as libero, choose the regular-rotation player they replaced so automatic swaps can work later.
  • When a paired libero rotates toward the front row, VolleyTrack can automatically swap the regular player back in without using a substitution.

Libero corrections

If a libero was accidentally entered as a normal starter, use Mark as Libero from the substitute sheet when the player is in a valid back-row context: P1, P5, or P6.

  • Libero correction does not decrement the substitution count.
  • Corrections while the set is still 0-0 should not create sub pairs.
  • If a libero is already on court, VolleyTrack should not ask whether another incoming player is also entering as libero.
  • Once a full rotation has passed, verify the current court state before changing libero identity.

Undo expectations

Undo should restore substitution count, libero identity, sub pairs, non-libero designations, rotation, score, serving team, and rally state for the action being undone.
