Lineup Corrections

Substitutions and Libero

Substitution logic keeps normal volleyball pairings fast while still allowing the coach to correct real-world lineup surprises.

Quick answer: Tap the on-court player to sub normally. Use ineligible overrides only when the real match is different from the app. Mark a player as libero only as a lineup correction, not as a counted substitution.

Normal substitutions

  1. Open the substitute sheet from the on-court player or lineup slot.
  2. Choose the eligible player entering the match.
  3. Confirm the substitution and continue scoring.

Ineligible players

VolleyTrack keeps likely-ineligible players grouped separately so accidental taps are less likely. If the real match requires it, expand that group and confirm the override.

Use overrides deliberately. Substituting an ineligible player is available for correcting reality, not for routine lineup flow.

Libero corrections

Only one libero should be on court at a time. If a libero was accidentally entered as a normal starter, use the libero correction action from the substitute sheet when the player is in an eligible back-row context.

  • Libero correction does not decrement the substitution count.
  • Corrections before the first point should not create sub pairs.
  • Once a full rotation has passed, verify the current court state before changing libero identity.

Undo expectations

Undo should restore substitution count, libero identity, sub pairs, non-libero designations, rotation, score, serving team, and rally state for the action being undone.
