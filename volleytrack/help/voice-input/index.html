Voice Input

Voice input lets you speak a rally description and review the parsed actions before they are added to the match log.

Quick answer: Tap the microphone, speak one clear rally, stop listening, review every parsed action, remove anything wrong, then confirm.

Voice input Premium

Speak one rally, review the parsed actions, then confirm.

Say short, structured phrases

Voice works best when you use player names or jersey numbers plus standard volleyball language.

  • "Kelsey receive good, Ella sets Harper for a kill."
  • "Number 7 kill from number 1."
  • "Opponent point" or "They hit out" for score-only situations.
  • "Timeout us" or "Timeout opponent" for timeouts.

Always review before confirming

  1. Check the stat type.
  2. Check the player and assist player.
  3. Remove incorrect actions before confirming.
  4. Use manual stat entry if the gym is too loud or the phrase is too complex.

Improve results in a noisy gym

  • Hold the phone closer than normal conversation distance.
  • Use jersey numbers when names are similar.
  • Speak one rally at a time instead of narrating multiple points.
  • Pause briefly before stopping so speech recognition has time to finalize the transcript.
Premium note: Free users can try voice input in a limited number of matches. Premium unlocks higher limits and removes ads.
