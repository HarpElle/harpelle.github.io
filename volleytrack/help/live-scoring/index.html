During The Match

Live Scoring

The live screen is designed for fast scoring first, then stat detail. Keep the scoreboard accurate and add detail when the rhythm of the match allows it.

Quick answer: Score each rally, log player stats from the stat controls or voice confirmation, use undo immediately for the last mistake, and use score correction for older errors.

Score points

  1. Tap the side that won the point.
  2. Confirm side-outs and rotation changes as the app advances service state.
  3. Use stat controls for aces, kills, attack errors, digs, blocks, receives, drops, and no-play situations.

Use undo correctly

Undo is intended for the most recent action. It should restore score, serving team, rally state, rotation, substitutions, timeouts, and libero/sub pairing state for that action.

Best practice: If the last tap was wrong, undo immediately. If the mistake happened several rallies ago, use score correction instead.

Correct the score

Use score correction when the scoreboard and real match diverge. This creates a deliberate correction without rewriting every previous event.

End sets deliberately

When a set reaches the target, choose the correct modal action: Keep Playing to close the modal only, Correct Score to fix the set score, Start Next Set to advance, or Finish Match to end the match.
