VolleyTrack Coach Test

VolleyTrack Match-Day Test Checklist

Use this during a real or scrimmage match. Preserve the score first, then send specific notes after the set or match.

Before the first serve

  • Install the latest TestFlight or internal test build.
  • Open VolleyTrack once before arriving courtside.
  • Confirm Premium/testing access if voice or summaries will be tested.
  • Create or choose the team, opponent, and match format.
  • Set lineup and confirm the first server.

Running the match

  • Score at least 10 rallies.
  • Use undo immediately after one intentional wrong tap.
  • Use score correction for an older mismatch.
  • Record at least one timeout.
  • Record at least one substitution or libero correction if it naturally happens.

For the stands

  • Start Fan mode from the live match.
  • Join from a second phone when available.
  • Confirm live score and recent activity update.
  • Verify any ad shown is a non-blocking banner.
  • Try one spectator alert or reaction if available.

After the final point

  • Confirm the final score before finishing.
  • Open the match summary.
  • Review player and team stats for obvious mismatches.
  • Generate or review an AI recap only after the score is correct.
  • Note app version, device, and match context.

Tell us what you found

  • What was the first confusing moment?
  • Was live scoring fast enough for match speed?
  • Did lineup, substitution, and libero handling match the real match?
  • Would voice input save time if recognition improves?
  • What would stop you from using VolleyTrack during the season?
