VolleyTally

The perfect companion for volleyball players, coaches, and fans

Download on App Store

Features

  • Track scores for multiple sets
  • Customize team names
  • Configure match settings (best of X sets, win by Y points)
  • Optional score cap setting
  • Dark mode UI for easy visibility in any environment
  • Simple, intuitive controls for fast score updates
  • Visual indicators for set and match winners
  • No internet connection required
